# StudentApp

Наданий проєкт реалізовує мобільний додаток, щоб полегшити життя студентам ХНУРЕ. Він допомагає відслідковувати крайні терміни здачі завдань, а також дотримуватися базової поведінки при роботі за комп'ютером, щоб зберегти своє фізичне здоров'я.

## Функціонал

### LOG IN
- Реалізований вхід користувача за допомогою email + password.
- Застосовано Firebase Auth.

### PROFILE
- Відображення відповідних даних, які відповідають певному студенту: ім'я, прізвище, група, спеціальність, номер студента та хобі.

### TASKS
- Відображення відповідних завдань для конкретного студента.
- Вказано назву роботи, короткий опис та крайній термін здачі.

### HEALTH
- Тут прописані базові заходи безпеки для здоров'я наших студентів.
- Реалізовано статичні поради, а також checkbox для зручного запам'ятовування вже зробленого.

### MAIN
- Розроблена навігація по всім попереднім екранам за допомогою методу Intent.

## Технології

- Мова: Kotlin
- UI: XML layouts
- Backend: Firebase
- Бібліотеки:
  - Firebase BOM: керування версіями бібліотек Firebase
  - Firebase Auth: система логіну та реєстрації
  - Firebase Firestore: хмарна база даних

## Встановлення

1. Клонувати репозиторій.
2. Налаштувати Firebase:
   - Завантажте файл google-services.json.
   - Вставте цей файл у папку app/ всередині проєкту (шлях: StudentApp/app/google-services.json).
   - В Android Studio натисніть кнопку Sync Project with Gradle Files.
3. Підключити реальний мобільний телефон або запустити емулятор.
4. Натиснути кнопку Run.
